# Business Buddy

This is a React project bootstrapped with [Vite](https://vite.dev/guide/) using [npm create vite@latest my-vite-project](https://vite.dev/guide/) for fast development and optimized production builds.

## Getting Started

Follow these steps to get your development environment set up and running:

### 1. Install Dependencies

First, you'll need to install the project dependencies. Run the following command in your terminal:

```bash
npm install
# or
yarn install
# or
pnpm install
```
### 2. Run Development Server

To start the development server, run:
```bash
npm run dev
# or
yarn run dev
# or
pnpm dev
```

This will start a local development server and open the project in your browser. By default, the fronted will be available at:

http://localhost:5173


You can start editing the app by modifying the files inside the src folder. The page auto-updates as you edit the files, so you can see changes instantly in your browser.
